Implementation Guide

1. HTML Structure

<div class="table-responsive">
    <table>
        <thead>...</thead>
        <tbody>
            <tr>
                <td data-label="Column Name">Value</td>
                ...
            </tr>
        </tbody>
    </table>
</div>

2. Key Points

  • Use data-label="Column Name" attribute on each <td>
  • Mobile (< 768px): Hides headers, converts rows to cards
  • Desktop (≥ 768px): Shows normal table layout
  • Each <td> displays label before value on mobile
  • Scroll hint appears on mobile tables

3. Responsive Classes to Use

  • table-responsive: Wrapper for table
  • action-btn: Buttons with proper touch size (44px+)
  • badge badge-success/warning/danger/info: Status badges
