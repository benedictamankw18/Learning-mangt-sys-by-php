Tailwind CSS Setup Complete

The LMS frontend now uses Tailwind CSS for styling

CDN Setup (Current)

Currently using Tailwind via CDN for quick setup. Perfect for development! 

<script src="https://cdn.tailwindcss.com"></script>
✓ Easy setup ✓ No build required ⚠ Not for production
Production Setup (Recommended)

For production, install Tailwind CLI for optimized builds. 

npm install -D tailwindcss
npx tailwindcss init
npx tailwindcss -i ./assets/css/input.css -o ./assets/css/output.css --watch
✓ Optimized ✓ Smaller files ℹ Requires Node.js

Quick Usage Guide

1. Add Tailwind to your HTML:

<script src="https://cdn.tailwindcss.com"></script>

2. Use utility classes:

<div class="bg-white p-6 rounded-lg shadow-md">
    <h2 class="text-2xl font-bold mb-4">Title</h2>
    <p class="text-gray-700">Content</p>
</div>

3. Use custom components:

<div class="card">
    <h2 class="card-header">Title</h2>
    <button class="btn-primary">Click Me</button>
</div>
